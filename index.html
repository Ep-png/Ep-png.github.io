O que é HTML?

HTML é a linguagem usada para criar a estrutura e o conteúdo de páginas na internet. É fácil de aprender!

Principais elementos do HTML

  • <h1> a <h6>: Títulos e subtítulos
  • <p>: Parágrafos
  • <ul> e <li>: Listas
  • <a>: Links
